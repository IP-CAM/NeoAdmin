Este template é uma alternativa ao template original do Opencart [www.opencart.com].
A proposta é ter uma opção mais intuitiva e agradável para quem gerencia uma loja baseada em opencart.
Este template foi desenvolvido utilizando o Framework Foundation 4 da empresa Zurb [http://foundation.zurb.com] e possui um layout responsívo e "mobile first", porém ainda serão feito muitos ajustes para um perfeito funcionamento em dispositívos móveis.

TESTADO SOMENTE NA VERSÃO 1.5.5.1 DO OPENCART.

ESTA É UMA VERSÃO BETA! USE POR SEU PRÓPRIO RISCO! NÃO É ACONSELHÁVEL O USO EM PRODUÇÃO NO MOMENTO.
Preferivelmente instale em uma instalação de teste e verifique se está tudo ok.

Você pode usar e modificar livremente este projeto, além de poder utilizar livremente em projetos comerciais. É vetado porém afirmar autoria deste ou a venda deste software em si.

INSTALAÇÃO


1. Envie todos os arquivos e pastas contidos na pasta "upload" para a pasta raiz da instalação do opencart a qual você deseja instalar o template.

2. Você será questionado se deseja substituir os arquivos. Clique em sim para todos.

3. Acesse a administração da loja e verifique se está tudo ok.

ATENÇÃO

Este template substitui quase todos os arquivos do template original do Opencart, se existir alguma alteração no template original, esta será sobreescrita. Nenhum módulo nativo deverá ter qualquer problema
, porém algumas alterações na admisnistração em si feitas via vQmod podem não funcionar pois algumas partes dos arquivos do template em si foram modificadas. Para que estes vQmods funcionem será necessário algumas adaptações.
